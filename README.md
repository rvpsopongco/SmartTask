# SmartTask

SmartTask is a simple task management application developed for a Software Design Laboratory. 
The project demonstrates the use of software design patterns and unit testing.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Categorize tasks (School, Work, Personal)
- Filter tasks by category or status
- Save and load tasks using interchangeable storage modules

## Design Patterns Used
- Singleton Pattern
- Strategy Pattern
- Factory Pattern
- MVC-inspired structure

## Repository Structure
SmartTaskApp/ - Main application  
SmartTaskTests/ - Unit tests  
diagrams/ - UML and Flowchart images  
docs/ - Design documentation
